# SUGAM

To install:
```bash
git clone https://github.com/mayuragarwal2004/sugam.git
cd .\sugam\client
npm i
cd ..\server\
npm i
```

To start React Development Server:
```bash
cd client
npm start
```

To run server(once):
```bash
cd server
npm run start
```

To run server(restart on change):
```bash
cd server
npm run dev
```
